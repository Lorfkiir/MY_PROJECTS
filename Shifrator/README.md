> Экспериментальный проект на Python для изучения принципов шифрования и создания собственного алгоритма преобразования данных.

О проекте

CRYPTO-BLOCK это учебный проект, созданный для изучения принципов работы шифрования и разработки собственных алгоритмов преобразования данных.

Идея проекта основана на принципах блочного шифрования: данные обрабатываются блоками и преобразуются с использованием ключа.

Алгоритм проекта не является копией существующих стандартов шифрования и не предназначен для защиты реальных данных. Это экспериментальная реализация для изучения криптографии.

Возможности

Приложение позволяет шифровать и расшифровывать текст с использованием пользовательского ключа, настраивать глубину преобразования данных и генерировать случайные ключи. Также реализована проверка надежности ключа, копирование результата, обработка ошибок, смена языка интерфейса и поддержка светлой и темной темы.

Максимальная глубина преобразования: 900 циклов.

Интерфейс

Графический интерфейс разработан с использованием CustomTkinter (Этим занимался chat-gpt :D).
